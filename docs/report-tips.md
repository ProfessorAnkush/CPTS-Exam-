# Quick CPTS Report Checklist (Use this before submit)

- [ ] Executive Summary < 2 pages, non-tech
- [ ] Every finding has: PoC command + full output + annotated screenshot
- [ ] Risk rating justified (not just "High because RCE")
- [ ] Remediation realistic & prioritized
- [ ] No spelling/grammar errors
- [ ] Table of findings with severity & CVE/CWE
- [ ] Professional tone throughout
- [ ] PDF under 20MB, readable fonts
